# Valquiz
quiz hvor man kan vite hvilken agent man burde bruke

# 13/05/2026

Jeg har tenkt til å lage et system i quizen der svarene ikke bare gir poeng til enkelte agenter, men i stedet bygger opp en spillprofil med egenskaper som aggressivitet, lagspill, kontroll og aim-fokus, og deretter er hver agent definert som en egen “profil” (for eksempel har Jett høy aggressivitet og aim men lav teamplay, Sage har høy teamplay og support men lav aggressivitet, og Sova har høy info/kontroll og medium i de andre), og til slutt sammenligner jeg brukerens profil med agentenes profiler ved å regne ut hvor stor forskjell det er mellom verdiene med en funksjon som summerer avstanden mellom hver egenskap, slik at agenten med minst forskjell blir valgt som resultat fordi den ligner mest på spillerens faktiske svar.

jeg har tenkt til å legge til en del hvor det står hvordan funker det? og gi tilbakemeldinger, for å vise bruketstøtte
